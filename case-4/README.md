# Кейс №4 — Web-приложение на Delphi

Каталог туров. WebBroker + ISAPI + MS SQL Server.

## Файлы
- `sql/schema.sql` — структура БД
- `app/` — Delphi-проект

## Запуск
1. Выполнить schema.sql в MS SQL Server
2. Открыть проект в Delphi 10.2, поправить параметры подключения в WebModuleUnit1.pas
3. Собрать ISAPI-модуль, подключить в IIS
4. Открыть в браузере

Анализ рынка WEB-систем — в answer.md.
