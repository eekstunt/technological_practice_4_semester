# Кейс-задача №1

Дан одномерный массив A размерности N. Найти сумму отрицательных элементов,
расположенных между максимальным и минимальным.

## Как решал

Сначала нахожу минимум и максимум за один проход, запоминаю их индексы.
Потом прохожу по элементам между этими индексами и складываю только отрицательные.

Если min и max стоят рядом — между ними ничего нет, возвращаю 0.

Сложность O(N) по времени, O(1) по памяти.

## Примеры

A = [5, -2, 1, -3, 4]
min = -3 (индекс 3), max = 5 (индекс 0)
Между ними: -2, 1. Отрицательный только -2, ответ = -2.

A = [1, 2, 3]
min = 1, max = 3. Между ними только 2, он положительный. Ответ = 0.
